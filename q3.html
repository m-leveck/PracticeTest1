Typesetting compared to Desktop Publishing

Old Fashion Typesetting

Old Fashion typesetting is a manual way to prepare and arrange text for printing.

Desktop Publishing

Desktop publishing is using a computer software to design things intended for print. For example, Adobe InDesign

The Use of HTML and CSS

HTML and CSS are used for websites. There are lots of customizable tools in HTMl and CSS that allow users to design user-friendly websites using coding languages.

Compare and Contrast

Old fashion typesetting is a manual process for print-only materials. For example, printing books, newspapers, and posters. It has limited flexibility and also takes a long time. Whereas desktop publishing is a software based approach for designing digital materials. This can include posters, magazines, and web design. Adobe InDesign is a great example of this. In desktop publishing its quick, easy, and much more flexible. CSS and HTML use coding languages to structure the design. It has more of focus on websites as opposed to other mediums like magazines, marketing materials, and books. However, it has great flexibility.
